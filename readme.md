# UIMSTest

## 关于UIMSTest

UIMSTest是个人应用，用于在JLU校内使用，方便查询校内通知、成绩、课程等，同时提供基础的管理功能。

应用主页： [UIMSTest网站](http://2045.site/)

## 近期更新

成绩：支持成绩搜索、支持使用学生VPN更新成绩

课程管理：添加、删除课程时间段

网页集成：通过js修改学生VPN主页item指向地址（可由此访问知网、万方等下载论文，查看校内通知等）


## 功能

课程、成绩、校内通知相关功能
